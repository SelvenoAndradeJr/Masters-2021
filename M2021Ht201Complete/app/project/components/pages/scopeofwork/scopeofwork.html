Scope Of Work

Here is your scope of work

For this exercise we are going to create a UI page that has two equally sized sections. The first sections will hold a D-pad object that will be created using CH5-button objects and styled using CSS to display as a traditional d-pad. The D-pad should show the icons for each button These icons can be found on the internet and are at the discretion of the person creating the interface. Once you have successfully styled your d-pad it will be converted to a reusable widget with dynamic values that can be placed in multiple places with minimal changes by the UI developer.

The right section will contain a CH5-list object that will be sized dynamically by the program currently running on your processor. There is a join schedule provided for you. Each button press shoud result in The list adding an item the first time each button is pressed and generating text letting you know that the button has been pressed and how many times it has been pressed since program initialization. All work should be done in the exercise page of this project.
