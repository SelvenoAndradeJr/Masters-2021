Course Outline

The topics below will be the learning objectives for this course

In this course we will cover the following:

  1. Designing and building a widget

    1. designing and styling objects on a single page

    2. those objects a widget (single instance of reusable and scalable code)

  2. Using lists

    1. dropping in the list

    2. connecting the relevant signals

    3. Styling Concerns (list vs. list items vs. template objects)

    4. Handling dynamic lists in programming

  3. Using the index property

    1. syntax and numbering

    2. Use within a widget

The courses Interactive Student Guide can be found in the zip file containing this package.
